# Epilepsy EEG AI — Papers + Reproducible Code

Honest subject-wise (LOSO) benchmarking of EEG-based epilepsy / seizure detection,
plus a systematic review. This repo holds the **papers** and the **Python code that
reproduces every number in them** — nothing else.

## Layout
| Folder | Contents |
|---|---|
| `code/paper/` | The three papers (`.tex` + `.pdf`) + `references.bib` — **edit in TeXstudio**, recompile to PDF |
| `code/reproducible/` | The Python that produces the paper's metrics (CHB-MIT LOSO, UCI epoch, multi-model, XAI, threshold, Siena) |
| `accuracy/` | The real computed metrics (`*.json` + `.md`) consumed by the papers |
| `architecture/` | Model card + data-split + normalization-stats configs |
| `data-samples/` | **10-row sample per dataset** (CHB-MIT, Siena) — illustrative only |
| `images/` | Figures generated by the analysis (confusion matrix, per-subject AUC, SHAP) |
| `model/` | Training entry point (no trained binary committed) |

## Papers (`code/paper/`)
- `review_full_2col.tex` — systematic review of EEG epilepsy AI + governed deployment
- `q1_full_2col.tex` — empirical Q1 paper **with** the RGAIG deployment framework
- `q1_noRGAIG_2col.tex` — the same empirical paper **without** RGAIG

```bash
texstudio code/paper/review_full_2col.tex   # or q1_full_2col.tex / q1_noRGAIG_2col.tex
```

## Datasets
- **CHB-MIT** Scalp EEG (`.edf`) — sample: 10 time-samples × 23 channels
- **UCI / Bonn** Epileptic Seizure Recognition (`.csv`) — sample: 10 rows
- **Siena** Scalp EEG / PhysioNet (`.edf`) — sample: 10 time-samples × 35 channels

> Samples are 10 rows for inspection only — not the full datasets (which are large +
> access-controlled). No patient-identifying clinical reports are included.

## Reproduce the paper's numbers (`code/reproducible/`)
Point the env vars / argv at your local copies of the public datasets, then run:

```bash
# CHB-MIT Leave-One-Subject-Out (raw EDF -> features -> RF/LOSO)
CHBMIT_DIR=/path/to/chbmit_edf python code/reproducible/chbmit_loso_pipeline.py

# Aggregate the per-fold results into the headline metric suite + per-subject stats
python code/reproducible/comprehensive_accuracy_analysis.py     # -> accuracy/comprehensive_metrics.json

# UCI epoch-level benchmark (the "epoch-level illusion" comparison)
UCI_CSV="/path/to/Epileptic Seizure Recognition.csv" python code/reproducible/verify_uci_epoch_level.py

# Multi-model LOSO comparison, XAI (SHAP), threshold tuning, Siena cross-dataset
python code/reproducible/multi_model_comparison.py
python code/reproducible/xai_feature_importance.py
python code/reproducible/improve_threshold_tuning.py
python code/reproducible/siena_loso_pipeline.py /path/to/siena_root
```

Each script writes its results under `accuracy/`. See `code/reproducible/README.md` for details.
All paths are env/argv-overridable (no hardcoded local paths); dependencies
(`numpy`, `scipy`, `scikit-learn`, `pandas`, `mne`) ship wheels for Windows and Linux.
